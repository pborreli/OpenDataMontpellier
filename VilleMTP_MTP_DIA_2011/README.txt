* ID : 103

* TITRE : Déclarations d’Intention d’Aliéner (DIA)

* IDENTIFIANT : VilleMTP_MTP_DIA_2011

* DESCRIPTION : Cette donnée renseigne les déclarations d’intention d’aliéner (DIA) de 2008 à 2011 pour la Commune de Montpellier. Lors de la vente d’un bien immobilier les notaires ont obligation d’informer la mairie des détails de la vente afin que la mairie puisse se substituer à l’acquéreur, elle utilise alors son droit de préemption. Cette donnée comporte les informations brutes dans le premier onglet, pour les suivants il s’agit d’un traitement statistique par type de bien, surface, type d’acquéreur, ou encore évolution des prix. 

Dans les versions Excels, sur l’onglet 3, il s’agit d’un tableau dynamique par quartier, il suffit de remplacer les cellules 1 E et F, par les cellules J et K correspondantes au quartier désiré. 

Ces données sont accompagnées par des traitements cartographiques de certaines données, compilés sous forme d’images dans un document PowerPoint.

* TITRE : Point adresse

* DESCRIPTION : Cette donnée renseigne les coordonnées géographiques pour chaque numéro de rue dans la commune de Montpellier. Cette donnée est généralement connue sous le nom de Point Adresse, elle est composée de 36.019 adresses géolocalisées. 

Certaines adresses ont pour numéro le zero (0, Rue du Mas de L'Entarayre), ce sont des adresses fictives : Créées automatiquement dès qu’une nouvelle voie a été numérisée Crées pour un besoin (géoréférencement d’un patrimoine ou autre qui n’a pas de n&#176; de voirie …). 

En général ces n&#176; de voirie nuls sont au milieu du segment

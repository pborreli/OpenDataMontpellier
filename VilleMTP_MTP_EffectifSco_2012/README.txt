* ID : 68

* TITRE : Effectifs scolaires

* IDENTIFIANT : VilleMTP_MTP_EffectifSco_2012

* DESCRIPTION : Cette donnée renseigne les effectifs scolaires pour les écoles maternelles et primaires de la commune de Montpellier. En plus des informations classiques comme le nom et l’adresse, pour chaque établissement le fichier renseigne le nombre d’enfants maximum, le nombre de classes et enfin le nombre de classes supplémentaires planifiées. 

Définition de certains champs : 

NBENFMAX : nombre d’enfant maximum 

NBCLAS : nombre de classe 

NBCLASFUT : nombre futur de classe 

ZEP : zone d’éducation prioritaire (oui/non)
